# Teste dos Arquétipos

Aplicação web em Next.js, TypeScript e Tailwind CSS para o **Teste dos Arquétipos**, desenvolvido por Clara Rangel.

O teste funciona 100% no client side, sem banco de dados, sem login, sem API paga e sem chaves externas. As respostas são salvas em `localStorage` para manter o progresso da pessoa durante a experiência.

## Tecnologias

- Next.js
- TypeScript
- Tailwind CSS
- Estado local no navegador
- `localStorage` para progresso e resultado

## Como rodar localmente

Instale as dependências:

```bash
npm install
```

Rode o servidor de desenvolvimento:

```bash
npm run dev
```

Acesse:

```bash
http://localhost:3000
```

## Scripts úteis

```bash
npm run dev
npm run build
npm run start
npm run typecheck
```

## Deploy na Vercel

1. Importe este repositório na Vercel.
2. Mantenha as configurações padrão de projeto Next.js.
3. Publique.

Não é necessário configurar variáveis de ambiente, banco de dados, autenticação ou serviços externos.
